Progressive Nav

Progressive responsive menu based on Flex and jQuery.

Download this project as a .zip file Download this project as a tar.gz file

According to the Nielsen Norman Group, the standard of using "hamburger" style menus on websites for mobile devices reduces usability. To solve this, this menu while in a mobile view, menu items on the right side of the menu are progressively "consumed" by a button that shows the number of hidden menu items.

Minimal

The code for this menu system is minimal. How minimal? Most responsive meny system have two sets of mark-up, one for the desktop version and one for the mobile version. The code for this component only has one set up mark-up. The only "hidden" item is the Menu button for the mobile view.

Flex Responsive

This menu system is fully responsive. The CSS for this menu is short because it utilizes the power of the Flex properties. This allows a fully developed menu system with far less CSS than would otherwise be possible.

jQuery

This menu system is compatible with jQuery libraries back to 1.7. If your website is already using jQuery, than you only need to add a few lines of javascript to make this menu system work.
